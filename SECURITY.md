# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in UrsaProxy, please report it privately by emailing:

**contact@alanbato.com**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond as soon as possible and work with you to address the issue before any public disclosure.

## Scope

UrsaProxy is a proxy that fetches content from Bearblog and serves it over Gemini. Security considerations include:

- Input validation for URL slugs
- Safe handling of external HTML content
- TLS certificate handling for Gemini protocol

## Supported Versions

Only the latest release is actively supported with security updates.
